This page is under development and has not been approved by the working group.

Implementation Summary

QualWeb version 0.7.20 is an Automated tool by LASIGE, Faculdade de Ciências da Universidade de Lisboa. This ACT implementation tests for WCAG 2.1 all levels, and WAI-ARIA 1.2. See understanding ACT consistency.
